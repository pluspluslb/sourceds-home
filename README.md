Source DS Home
===========
scripts and configs for tf2, csgo, and insurgency on source dedicated server

**get started**

sudo adduser sourceds

sudo su - sourceds

git init

git remote add origin https://github.com/pluspluslb/sourceds-home.git

git fetch

git reset --hard origin/master
